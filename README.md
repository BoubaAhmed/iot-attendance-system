# Système de gestion de présence IoT

Projet de Master visant à concevoir un **système intelligent de gestion de présence automatisée** basé sur l’Internet des Objets (IoT).

## Objectif

Automatiser l’enregistrement des présences à l’aide de la biométrie (empreinte digitale) et offrir une interface web pour la visualisation et la gestion des données en temps réel.

## Technologies utilisées

- **ESP32** – microcontrôleur IoT
- **Capteur d’empreintes digitales** – identification biométrique
- **LCD** – affichage local
- **Flask (Python)** – API backend
- **Firebase** – base de données temps réel
- **React.js** – interface web (dashboard)

## Fonctionnalités

- Identification des étudiants par empreinte digitale
- Enregistrement automatique des présences
- Synchronisation des données avec Firebase
- Tableau de bord web pour la gestion et le suivi
- Surveillance de l’état des ESP32 (en ligne / hors ligne)
